MCP Server: Model Context Protocol Server 

Tools – functionality we give the model
ex: convert_product_data_to_ebay(shopify_product_data)

Tool calls – requests from the model to use tools
ex: 

Tool call outputs – output we generate for the model

Functions versus tools (Định nghĩa sâu về schema input và output -> số lượng token lớn)


Tool: function tool 
Tool call: request use function tool 
Tool call output: execute function tool return output response 
Function: một loại định nghĩa bằng json schema để đảm bảo input/output rõ ràng 

Flow:
1: Make request to model with tools 
2. Receive tool call from model 
3. Execute code 
4. Tool output 
5. Final response
